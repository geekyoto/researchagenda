This is researchAgenda a site to post the ideas that you are interested in researching. 

By sharing this idea with your followers on Twitter they can feedback, suggest if the idea is interesting, add comments and also point you to initial reading or other work that you should be aware of before starting the research.

The site is written using:

* Node.js
* MongoDB
* Twitter Bootstrap

It makes use of the following Node libraries:

* Express.js
* Jade
* Mongoose
* Passport
* Passport-Twitter
* Connect Ensure Login
